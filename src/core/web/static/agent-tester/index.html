This application allows you to test and interact with MCP servers through an AI agent.

Getting Started

  1. Configure an MCP server connection in the sidebar
  2. Specify HTTP header values, if required
  3. Select the model and specify the parameters
  4. Correct or set an agent (system) prompt if needed
  5. Set custom prompt if needed
  6. Start chatting with the AI agent

The agent will use the connected MCP server's tools!
